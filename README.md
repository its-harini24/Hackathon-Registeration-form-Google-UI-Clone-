# Google Form Clone

A responsive Google Form Clone built using HTML, CSS, and JavaScript. This project recreates the basic design and functionality of Google Forms, allowing users to interact with a form interface.

## Features

- Responsive user interface
- Google Forms-inspired design
- Form validation
- Input fields (Text, Email, etc.)
- Submit button
- Clean and user-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/its-harini24/google-form-clone.git
```

Navigate to the project folder:

```bash
cd google-form-clone
```

Open `index.html` in your browser.
